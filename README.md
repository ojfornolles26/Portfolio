# Orlando Fornolles Jr. — Professional Portfolio

An editorial-inspired, minimalist digital resume and portfolio website showcasing front-end engineering, local-first web applications, and AI integrations. Designed for maximum responsiveness, visual elegance, and fluid layout flow using a clean Bento grid pattern.

---

## 🎨 Featured Projects

The portfolio showcases three core web applications built with a privacy-first, client-side architecture:

### 1. [Filo](https://github.com/ojfornolles26/Filo) (Web App / Productivity)
*   **Description**: A browser-native document workspace for batch image-to-PDF conversion, lossless PDF exporting, and high-performance client-side PDF-to-image/text extraction. 
*   **Key Features**:
    *   100% private local processing (files never leave your device).
    *   Lossless PDF export with adjustable quality control.
    *   Batch PDF-to-image extraction with JSZip packaging.
    *   Structured Markdown text and metadata parsing using `pdfjs-dist`.
*   **Stack**: React, Vite, Tailwind CSS, jsPDF, PDF.js, JSZip, Motion.

### 2. [Signity](https://github.com/ojfornolles26/Signity) (Web App / Productivity)
*   **Description**: A privacy-first digital signature tool with dual creator modes for hand-drawn signing and typed calligraphy signatures.
*   **Key Features**:
    *   Dual hand-drawn (stylus/mouse/trackpad) and typed cursive signature creation.
    *   High-precision curve smoothing using quadratic curve interpolation.
    *   6 calligraphic/cursive font styles with size and formatting controls.
    *   Pixel-based canvas scanning to auto-crop signatures for transparent PNG export.
*   **Stack**: React, Vite, Tailwind CSS, HTML5 Canvas, Google Fonts, Motion.

### 3. [Embraze](https://github.com/ojfornolles26/Embraze) (Web App / Community Safety)
*   **Description**: A community-driven safety and hazard reporting platform featuring an interactive map of Cebu City.
*   **Key Features**:
    *   Interactive Leaflet mapping system.
    *   Community verification system with hazard dispute and upvoting.
    *   Auto-expiry controls for localized hazard pins.
*   **Stack**: React, Vite, Leaflet, Tailwind CSS, Lucide React.

## 💻 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation
1. Clone the repository.
2. Install the package dependencies:
   ```bash
   npm install
   ```

### Development Server
Run the local next development server:
   ```bash
   npm run dev
   ```

### Production Build
Create the optimized static production build:
   ```bash
   npm run build
   ```
